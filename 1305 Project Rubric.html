Rubric for CIST 1305 Projects

Link Back to Homework Page

Overall Program  (25%) Score
     Program is Easy to Read and Understand  
     Indention and White Space are used well  
     Loops are used properly and efficiently  
     Conditional Statements are used properly and efficiently  
     Variables are Properly Defined  
     Algorithm/Logic is Efficient  
Internal Documentation  (20%)  
     Program has a Header with student name/date/program name  
     Explanatory comments are included with complex code  
     Objects must be named according to standards  
     Variable names are descriptive  
Input and Output  (20%)  
     Good Use of Prompts for User Input  
     User Instructions are Provided as Needed  
     Output is Labeled Clearly  
     Program adheres to standard Windows user interface  
     Input and Output are Neat, Well Formatted, Not Cluttered  
Program Execution  (35%)  
     Program runs without errors  
     Program meets the specifications of the problem  
     Program correctly solves the problem  
Other  (Up to 5 extra points)  
     Extra features (must be unique)  

Each of the statements in the rubric above will be rated using a scale of 0 to 5 where:

0 represents completely false
1 represents just barely true
2 represents somewhat true but mostly untrue
3 represents just about half right
4 represents mostly true
5 represents completely true

Students are encouraged to share ideas and get assistance from one another, but each student must submit his/her own unique solution to the problem. Any students submitting work that is too similar will receive the credit for the work distributed among those students. Team work must be cited and credits assigned in documentation.

All programs should be turned in on time.   Programs will lose points by being late.
