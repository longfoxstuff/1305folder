 Focal Point    Flow Control - Sequence
 Learning    Sequence is the straightforward execution of one processing step after another.
 In pseudocode, we represent this construct as a sequence of pseudocode statements,
 see below:

Statement 1
Statement 2
Statement 3

The sequence control structure can be used to represent four basic computer operations: to receive information, put out information, perform arithmetic, and assign values.
In an algorithm it would look like the example below:

Prompt for item cost
Calculate tax
Print item cost
Print tax

As you can see, a sequence control structure is a straightforward list of steps written one after the other, in a top-to-bottom fashion.
Each instruction will be executed in the order in which it appears.
