CIST 1305 Lesson 19 Strings

 Preparation
 for Learning		    Steps:
  • A string of ________
 Focal Point    Elements of Programming - Strings
 Learning  

 Definition: A string is a group of characters.

 Definition:  A literal string is a group of characters surrounded by quotation marks

 String variables in JavaScript use quotes around the value.

 You assign a value to a string variable just like any other variable.

     Examples:

  • var txt="Hello world!";
  • document.getElementById("demo1").innerHTML = txt.length);
  • var lastName ="Plong";
  • document.getElementById("demo2").innerHTML = lastName);

 Contents of string variables can include basically any printable character
 and even some non-printables.

 Caution!!!  A string of numbers is not a number.
 (A classic example of a string of numbers is a phone number.)

  • num1 ="25";
  • num2 =25;
  • document.getElementById("demo1").innerHTML = "num1" * 2);
  • document.getElementById("demo2").innerHTML = num2 * 2);
