 Learning    Data Types:

 The four Data Types in JaveScript are: Number, String, Array, and Object.

  • What data type would the number 7 require?
  • What data type would a social Security Number require?
  • What data type would "Volvo" require?
  • What data type would the number 1,899,234,210.2 require?

 The four data types used in JavaScript can be found on the W3Schools web site.

 Some special characters and words cannot be used in JavaScript.
 See W3Schools, JavaScripter.net, and Global Guide Lines for lists.

 The preferred way to set up variables for use is with the VAR statement.
 It is good programming practice to declare all variables before they are used.
 They should be defined with the VAR statement early in the program.
 zPreferrably immediately after the program header.

 The syntax of the VAR statement is quite simple. See below:

 Examples of the four data types:

1. Number Example:

This sets up a variable as a number;
assigns the variable a specific value;
and stores it in memory.

var x = 4; var y = 8; var z = x + y;
x stores the value 4 x stores the value 8 z stores the value 12

2. String Example:

This sets up a variable as a string of text;
assigns the string a specific value;
and stores it in memory.
Strings are used for manipulating and storing text.

var schoolName = 'Our school name is \"NGTC\".'; var schoolMascot = "We used to be the \"Eagles\", but now we are the \"Wolves\".";

3. Array Example:

This sets up a set of variables as an array;
assigns the set (array) a specific value;
and stores it in memory as a single variable.

var autos = ["Chevrolet", "Chrysler", "Ford", "Jeep"];

4. Object Example: Objects can be single (simple value) or plural (many values).

An item is an object. An object as a simple value variable;

var auto1 = "Ford";

An object can contain many values. An object with many values is
     written as name:value pairs;
This sets up an object container;
places the named values inside the container;
assigns a name to the container (object);
and stores it in memory as a single variable called an object:
Objects are containers for named values.

var auto = {manufacture:"Ford", model:"Edge", year:"2010", color:"blue"};

This statement could be used to set up a counter variable which would only
hold small integer values. (NOTE:  Please spell INTEGER correctly!!!)

 VAR statements are not limited to defining one variable at a time:
 they can be used for multiple variables
