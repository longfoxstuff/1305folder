 Focal Point    Elements of Programming - Variables
 Learning    Mathematical definition:  A variable is a quantity that can assume any value.

 Computer definition:  A variable is a named location in memory where values are stored.
 These values can be changed throughout a program's execution.

  • Think about variables at temporary holding areas for information
  • How long will these holding areas be available?
  • What does the word initialize have to do with variables?
  • What does the word initialize have to do with a computer's memory?

 Naming variables:

  • Use descriptive names
  • Only use alpha-numeric characters for names of variables
  • The name must begin with a letter or underscore
  • JavaScript is case consistent
  • NEVER use a JavaScript keyword

 Using variables in JavaScript:

  • As a good general rule always define your variables up front using the VAR statement
  • When giving a variable a value keep the variable to the left of the equal sign (example num = 3)
  • The data to the right of the equal sign will be the value of the variable when the line of code has been interpreted
  • Think of the equal sign as meaning "is assigned the value of" instead of "is equal to" as in mathematics
