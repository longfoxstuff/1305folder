CIST 1305 Lesson 09 IPO Tables

 Preparation
 for Learning		    Steps:
Now that you've looked at some decision making processes and some linear processes, we're going to think about a few more parts of this process:
  • How did you know there was no milk in the fridge?
  • How did you know your mom, dad, spouse, other was upset?
  • How did you respond to these actions?  Verbally?  Body language?  Actions?
  • Think about these as inputs and outputs
 Focal Point    Planning Your Program - IPO Tables
 Learning    Definition - An IPO Table or Defining Diagram identifies the inputs to a program, the outputs the program will generate, and the processing steps required to transform the inputs into the outputs.

 Example:

Input Processing Output
Unit Price Read unit price, quantity purchased, discount code Net amount due
Quantity Purchased Calculate gross amount due  
Discount Code If discount applies, calculate discount amount  
  Calculate net amount due  
  Print net amount due  

To Determine the Inputs and Outputs you should look for nouns and adjectives as you read the problem.  Verbs will most likely be processes.

 Example:

Note: In this example, Nouns/adjectives are grey  and verbs (actions/processes) are blue

A program is required to read three numbers, add them together, and print their total.

Input Processing Output
number_1 Read three numbers total
number_2 Add numbers together  
number_3 Print total number  
     
     

  Group Work, Problem 9-1: (Corresponds to Lesson 10-1; Programming with Flowcharts)
       Construct an IPO Table for the following problem:
       You will instruct the user to input their name and three characters;
       Receive the characters from the user;
       And display the following message:
       "Hello (name)! Have a nice day".

  Individual Work, Problem 9-2: (Refer to Lesson 10-2 IPO Tables Individual Work)
       Construct an IPO Table for the following problem:
       You will receive two integers (whole numbers) from the user;
       You will add the numbers, subtract the numbers, multiply the numbers, divide the numbers;
       And then display (print) the results on the screen.
