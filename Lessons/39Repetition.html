Focal Point . Flow Control - Repetition
Learning .

Repetitive structures as we've discussed them so far have all relied on user input or programmed-in values to start and stop our loops.  Sometimes we don't have that luxury.  We have to use sentinel values to trigger the end of our loop.

Write a program to add up all the numbers as entered by a user.  When the user enters 999 you stop the program and print the sum.  Do not add the 999 to the total.

REM plong
REM 6-5-2005
REM Program to demonstrate the use sentinel values

DIM num AS SINGLE, total AS SINGLE

CLS

PRINT "End the program by entering '999'"
PRINT
INPUT "First number"; num

DO WHILE num <> 999
   total = total + num
   INPUT "Next number"; num
LOOP

PRINT
PRINT "Your total is"; total

END
