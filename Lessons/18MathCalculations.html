 Focal Point    Elements of Programming - Mathematical Calculations
 Learning    Mathematical Calculations in JavaScript can be done simply with numbers:
  • document.getElementById("demo").innerHTML =(7 + 3);          Result will be 10
  • document.getElementById("demo").innerHTML =(4.3 - 5);        Result will be -.7

 Mathematical Calculations in JavaScript can include variables:

  • x = 4.3
  • y = 5
  • document.getElementById("demo").innerHTML =(x - y);           Result will be -.7
 Assignment    Write a HTML document using JavaScript to calculate the following numbers.
 Name it 18-1.html.

  • 7 + 2

  • 72 * 41

  • 3 - 459

  • 2 / 14

  • 2 ^ 3   hint: Math.pow(2, 3)
