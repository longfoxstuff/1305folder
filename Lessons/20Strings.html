 Learning    Defining a String:
  • A cleaner way to define a string is to use the var statement
  • This can be used in two different ways as follows:

var mystr
var str2 = "1234567"
mystr = "1234567"
document.getElementById("demo1").innerHTML = (mystr);
document.getElementById("demo2").innerHTML = (str2);

  • This program's output will be the following:

1234567

1234567

 Combining Strings (concatenation):

  • Two strings can be combined to form a new string consisting of the strings joined together.  The joining operation is called concatenation and is represented by an plus sign (+)
  • For instance, "good" + "bye" is "goodbye"
  • A combination of strings and ampersands that can be evaluated to form a string is called a string expression
  • The assignment statement and PRINT statement evaluate expressions before assigning them to variables or displaying them
  • The following program illustrates concatenation:

<!DOCTYPE html>

 Additional:

  • The string "", which contains no characters, is called the null string or the empty string.  It is different from " ", the string consisting of a single space.  String variables that have not been assigned values initially have "" as their default values.  (Numeric variables have default value 0.)
  • Assigning a string value to a numeric variable can result in an error.
  • The quotation-mark character (") can be placed into a string constant by using &#34;
  • For example, when the statement

            document.getElementById("demo").innerHTML =
            "George" + &#34; + "Babe" + &#34; + " Ruth"

    is executed, the following is displayed:
        George "Babe" Ruth
