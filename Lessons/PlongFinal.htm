CIST 1305 Final Exam

 

Write a program that will do the following:

A computer store sells diskettes at 50 cents each for small orders or 30 cents each for orders pf 25 diskettes or more. Write a program that requests the number of diskettes ordered and displays the total cost. Test the program for purchases of 5 and 30. (The program should yield $2.50 and $9.00 respectively.)

You are required to develop and turn in the following: an IPO table, a Flowchart (hand written is fine), an Algorithm and the Qbasic code for this assignment.

To receive full credit for this assignment all 4 items must be turned into the instructor.
