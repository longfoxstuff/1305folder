Where did Programming Languages start?
  • Dr. John von Neuman's paper on the stored program concept 1945
  • The first computers were programmed with wires, not with real languages at all
  • There are now 5 major categories of programming languages
   1. Machine Languages
  • Written to run on one particular design of computer
  • Use binary or hexadecimal digits to represent instructions
  • Tedious and time-consuming
   2. Assembly Languages
  • Written to run on one particular design of computer
  • Uses symbolic instruction codes, also called mnemonics
  • Must be translated into machine code before the computer can understand it
  • This is done by an assembler
   3. Third-Generation Languages
  • Machine Independent
  • Written as a series of English-like words
  • Much easier for a programmer to write than Assembly or Machine Language
  • Must be translated into machine code before the computer can understand it
  • This is done by a compiler or an interpreter
  • These languages are generally structured telling the computer what to do and how to do it
   4. Fourth-Generation Languages
  • Machine Independent
  • Written as a series of English-like words
  • The difference is that 4GL's only specify what to do, not how it should be done
   5. Natural Languages
  • Machine Independent
  • Does not follow a specific set of rules and syntax
  • Allows the user to enter requests that resemble human speech
  • Often associated with artificial intelligence
  • Not yet popular in business apps
