CIST 1305 Lesson 13 Pseudocode

 Preparation
 for Learning		    Steps:
  • We have used two techniques so far to help us think about programming: IPO Tables and Flowcharts
  • IPO tables are word driven but don't allow for flexibility in the flow of a problem
  • Flowcharts are graphical and allow for total flexibility in the flow of a problem, but don't allow for deep explanations of processes.  The graphics can be a challenge in and of themselves
  • Is there a mechanism which would allow us room for plenty of words of explanation while giving us flexibility of flow and not hampering us with memorizing special graphical characters?

 Focal Point    Planning Your Program - Pseudocode
 Learning    Definition:
 An Algorithm is a series of steps, or a method,
 for converting the inputs for a specific problem into a meaningful output;
 it is a "recipe" for problem solving
  • Could you follow these directions?  Could you write directions to cook your favorite dish?
  • To write an algorithm, such as a recipe, you would probably use English statements
  • Programmers often use a "special" language to write algorithms called Pseudocode

 Definition:
 Pseudocode is "false"  code consisting of English-like phrases
 used to develop or describe an algorithm

  • You can think of pseudocode as a kind of shorthand

 Example: Add Three Numbers

 A program is required to read three numbers, add them together, and print their total.

 IPO Table

 number3

 Input number2

  total

Flowchart

    

 Pseudocode

 addThreeNumbers

    Input number1

    Input number2

    Input number3

    total = number1 + number2 + number3

    print total

 end

 Group Work:  Create a flowchart and an algorithm to find the average temperature:

  • Prompt user for minimum and maximum temperature for any given day
  • Add temperatures as intergers together and divide by 2
  • Display average temperature

 Individual Work:  Create a flowchart and an algorithm to compute mowing time:

  • Prompt user for lot width and lot length
  • Prompt user for house width and house length
  • Compute area to be mowed
  • Compute time to mow lot at two square yards per minute
  • Display mowing time per minute
 Assignment    Create a an algorithm (pseudocode):

  Problem 13-1: (Corresponds to 9-1 and 10-1)
       Construct an algorithm for the following problem:
       You will instruct the user to input their name and three characters;
       Receive the characters from the user;
       And display the following message:
       "Hello (name)! Have a nice day".

  Problem 13-2: (Corresponds to 9-2 and 10-2)
       Construct an algorithm for the following problem:
       You will receive two integers (whole numbers) from the user;
       You will add the numbers, subtract the numbers, multiply the numbers, divide the numbers;
       And then display (print) the results on the screen.

  Problem 13-3: (Corresponds to 9-3 and 10-3)
       Construct an algorithm for the following problem:
       You will receive an integer (whole number) from the user;
       You will add 2 to the number, triple the number, subtract 5 from it;
       and then display (print) the result on the screen.
