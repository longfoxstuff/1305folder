 Focal Point    Elements of Programming - Variables & Constants
 Learning    Variables - From whence cometh my value?
  • Variables by their very nature don't have their values programmed into the code
    (except perhaps their initial value)
  • They have to come from some outside source else they would not be "variable"
  • One way to get a value is to read it from a data file
  • Another way would be to read some kind of input device for a value
    (such as an Analogue to Digital Converter)
  • One of the most common ways to assign a value to a variable is to receive it from the user
  • JavaScript uses a Prompt statement to do that

 INPUT'ting values into Variables

  • Reads input from the keyboard or a file.
    We're concentrating right now on the keyboard aspect of it
  • The syntax on this command is as follows:
  • prompt("sometext","defaultvalue");
  • Practical application is as follows:
  • var name=prompt("Please enter your name"," ");
  • This will give the variable name the value the user keys into the computer
  • Once this value is in the variable it can be manipulate in any way desired and then output in any way desired

 PRINT'ing variable values to the screen

  • PRINT is a flexible command able to write information to the screen or to a file
  • We will mainly be concentrating at this time on writing information to the screen
  • The syntax of the PRINT command when used for screen display is as follows:
  • document.getElementById("demo").innerHTML =
  • Practical application is as follows:
  • document.getElementById("demo").innerHTML("Hello " + name + "! How are you today?");
  • This will display on the screen the words Hello "name"! How are you today?

 Some things just shouldn't change....

  • There are some times when you don't want to change a number under any circumstances barring a major rewrite of the program
  • An example of this might be the value for pi
  • You might not want to calculate it every time (UNDERSTANDABLY SO!) so that might be a variable you want to set up to use in your calculations
  • But since the variable doesn't change, it is actually a constant
  • The syntax for the CONST command is as follows:
  • CONST constantname = expression
  • Practical application is as follows:
  • CONST PI = 3.141593
  • This would create a memory location referenced by the name PI containing the value 3.141593
  • You cannot change the value of PI after defining it
  • You cannot VAR a variable named PI in the same program as creating it as a constant
