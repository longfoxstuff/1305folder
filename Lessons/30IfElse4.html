 Homework    Be sure to look at my program for an idea

 1. Write a program to receive two numbers from the user.
 Divide the first number by the second and report the answer.
 Please consider the special case, however, that no number can be divided by zero.
 In that case give the user a message that a zero divisor is invalid.

 2. Write a program to receive one number from the user.
 Calculate the square root of that number.
 However, if the number is negative please give the user a message that the answer will be imaginary.
 Make sure your program doesn't crash!

 3. Write a program to receive the name of an animal.
 That name can be cat, dog, mouse, elephant, tiger, or cow.
 Respond to the user with a statement describing the animal,s sound,
 Example: The cat says MEOW.

 Submit assignment to your student folder on the X Drive.
 Please include the following information:
 Your Name, Course, Lesson Number and Assignment, and Email Address.
