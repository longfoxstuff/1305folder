Focal Point . Flow Control - Repetition
Learning .

The INKEY$ function is very useful in writing programs when you need to interact with the user.  The INKEY$ function reads the keyboard once and continues on.  When used in a loop it will stop the program from continuing until the user presses an appropriate key.

Check out the discussion about INKEY$ on the QBasic.com tutorial page.
