 Learning  

 Remember the symbols for flowcharts.
 Those symbols can directly relate to certain phrases which may be used in IPO charts.
 These are some examples.

Symbol Use Examples
Terminal  This can be a header or the word "end"

    rem This is the beginning of the program

    END
Process  x = 7 + 4

 total = subtotal + tax
Decision  if x = 11 then ...
Input/Output  input "Please type your name", name$

 print "Hello "; name$

 Assignment    Construct a flowchart for the following:

  Problem 10-2: (Corresponds to Lesson 9-2; IPO Tables)
       Construct a Flow Chart for the following problem:
       You will receive two integers (whole numbers) from the user;
       You will add the numbers, subtract the numbers, multiply the numbers, divide the numbers;
       And then display (print) the results on the screen.

  Problem 10-3: (Corresponds to Lesson 9-3; IPO Tables)
       Construct a Flow Chart for the following problem:
       You will receive an integer (whole number) from the user;
       You will add 2 to the number, triple the number, subtract 5 from it;
       and then display (print) the results on the screen.
