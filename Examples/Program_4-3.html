CIST 1305

String Comparison Program

A simple program which allows a string comparison:
In this case, it compares an entered password with the correct password (A7prospero6).

Click on PasswordChecker to demonstrate prompt box.

Run as is with PLong; run with correct password A7prospero6; also try running using other passwords.

Results:

Pseudocode:

1 // A variable to hold a password.
2 Declare String password
3
4 // Prompt the user to enter the password.
5 Display "Enter the password."
6 Input password
7
8 // Determine whether the correct password
9 // was entered.
10 If password == "A7prospero6" Then
11 Display "Password accepted."
12 Else
13 Display "Sorry, that is not the correct password."
14 End If

Expected Program Output (with Input Shown in Bold):

(This program can give two different outputs depending on input.)

(Output Option 1)
Enter the password.
PLong [Enter]
Sorry, that is not the correct password.

(Output Option 2)
Program Output
Enter the password.
A7prospero6 [Enter]
Password accepted.
