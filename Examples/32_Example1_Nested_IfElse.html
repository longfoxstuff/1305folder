CIST 1305, Lesson 32 Example: Nested If Else

Program to Process Customer Record

Results

 

Program requirements:

  • Get customer's name
  • Get customer's purchase amount
  • Get customer's tax code

Tax Codes:

  • 0 tax exempt (0%)
  • 1 state tax only (3%)
  • 2 federal and state sales tax (5%)
  • 3 special sales tax (7%)

Program must calculate:

  • The sales tax
  • Total amount due

Program must print (output):

  • Customer's name
  • purchase amount
  • Sales tax
  • Total amount due

IPO chart:

 Input  Processing  Output
 custName  calculate salesTax  custName
 purchAmt  calculate totalDue  purchAmt
 taxCode    salesTax
     totalDue

Algorithm:

	Process_customer_record
	   Get custName, purchAmt, taxCode
		IF taxCode = 0 THEN
		  salesTax = 0
		ELSE
		  IF taxCode = 1 THEN
		   salesTax = purchAmt * 0.03
		  ELSE
		   IF taxCode = 2 THEN
			 salesTax = purchAmt * 0.05
		   ELSE
			 salesTax= purchAmt * 0.07
		   ENDIF
		 ENDIF
	   ENDIF
	   totalAmt = purchAmt + salesTax
	   Print custName, purchAmt, salesTax, totalDue
	END
